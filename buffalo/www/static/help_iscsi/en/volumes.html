[]Set Up Volume
Click [Add Volume] to show the Add Volume screen.
Click Volume Name to edit volume information.
Click [Delete Volume] to delete a selected volume.
The following operations will display a confirmation screen:
- Build or change a RAID array
- Remove Volume
- Set up Logical Volume Manager (LVM) - Restoring default settings
- Erase disk
Enter the displayed number in the confirmation field within 60 seconds, then click [Apply].

[]Add Volume
Volume Name
Volume names may contain up to 12 alphanumeric characters. Letters, numbers, and hyphens (-) may be used. The first character in the name should be a letter, not a number or hyphen.
Volume Description
Volume descriptions may contain up to 50 alphanumeric characters. Letters, numbers, hyphens, underscores, and spaces may be used.
Disk Area
Select disk area to create volume.
Size
Enter volume size.
Access Control (Each volume individually)
Select [Enable] to use access control function.
The user name and password are required for the iSCSI HDD Connecting Tool to connect with the TeraStation. If [Password(Mutual Authorization)] is set, select [Use Mutual Authorization] when the authorization dialog appears. Specific IP addresses are allowed by [IP Address Limitation]. Enter IP addresses separated by commas. For example, "192.168.11.1,192.168.11.2". If only one IP Address is allowed, enter it with no comma. If left blank, all IP address are allowed. Entered IP addresses are allowed access.
Advanced (Each volume individually)
Default settings are recommended for these settings. If your settings prevent you from connecting, Click [Restore Default Settings] to restore settings to factory defaults.
