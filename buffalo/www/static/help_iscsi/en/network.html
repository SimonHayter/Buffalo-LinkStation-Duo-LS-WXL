[] Ethernet Frame Size
Data transfer efficiency can be improved by changing the maximum value of data size that can be transmitted at once.
- 1,518 bytes (default) : It is set as 1,518bytes in default
- 4,102 bytes (Jumbo Frame) Transfer with 4,102bytes
- 7,422 bytes (Jumbo Frame) Transfer with 7,422bytes
- 9,694 bytes (Jumbo Frame) Transfer with 9,604bytes

[] Port Trunking
Port Trunking lets you connect Ethernet port 1 and Ethernet port 2 in parallel for higher throughput and increased redundancy.
