[] Group Management
Group Settings
A list of groups on the device is displayed. To create a new group, click [Create Group].
To edit a group, click on its group name, and a screen to edit group properties opens.
To delete any group, select (check) the group and click [Delete].

[] Add New Group
Group Name
- Up to 12 alphanumeric character can be entered.
- Alphanumeric characters, `-`(hyphen), `_`(under score) and `.`(dot) may be used.
- Symbols (except for `_`(under score)) cannot be used as the first character.
Group Description
Enter a description of the group.
- Up to 50 alphanumeric characters (up to 25 two bytes characters) can be entered.
- Alphanumeric characters, two byte characters, `-`(hyphen), `_`(under score), and ` `(space) may be used.
- Symbols (except for `-`(hyphen) and `_`(under score)) cannot be used as first character.

[] Group Members
Specify users who should belong to the group.
Domain users assigned by the domain controller cannot be added to a group on the LinkStation.
If User Descriptions are entered, a description of each user will be displayed.

[] Domain Group List
Domain Group List
If participating in a domain, all domain groups will be listed here.
