[] Restore Factory Defaults
Clicking [Restore LinkStation] restores your LinkStation to original factory settings. You may choose the behavior of the reset button with regard to the Admin Password on the LinkStation. If [Keep current admin password] is selected, then the admin password will not be reset. If [Restore admin password to factory defaults] is selected, then the [Restore LinkStation] button will reset the admin password to 'password'.

[] Erase
Click [Erase] to overwrite all data on the disk with zeros. Erasing the LinkStation is recommended when you sell, donate, or give it away, or if you otherwise want to delete all data from the unit so that it can never be recovered. This cannot be undone. After erasing, all settings on the LinkStation are restored to their factory defaults.
