[] User Management
Click [Create User] to add additional users.
Click on a user's name to edit that user's settings.
To delete a user, put a check next to the user's name and click [Delete].

Notes: -Admin and Guest are built-in user accounts. They cannot be edited or deleted.
If additional users access the LinkStation through a domain, then an additional list of users ("Domain User List") will also be displayed.

[] Create User
Username
To add a new user, enter a username and password. A description for the user may also be entered.
Notes:
-Names and Passwords may contain up to 20 alphanumeric characters.
-User descriptions may contain up to 50 alphanumeric characters or up to 25 two-byte characters.
-Alphanumeric characters, `-`(hyphens), `_`(underscores), and `.`(dots) may be used in names.
-Alphanumeric characters, `-`(hyphens), and `_`(underscores) may be used in passwords.
-Alphanumeric characters, two-byte characters, `-`(hyphens), and `_`(underscores) may be used in descriptions.
-Numbers and signs cannot be used as the first character in a name.
-Windows98se/98/95 users should not use more than 14 characters in the password. Mac OS 9 (and earlier) users should use no more than 8 characters in the password. Longer passwords will make it impossible to access the LinkStation from these machines.
-New users are added to the hdusers group. You may modify group membership in the Group Management section of this GUI.

External Authentication User List
