[] Media Server
Media Server Settings
To change settings, click [Modify Settings]. If your media player requires a media server to play media from your LinkStation or TeraStation, then enable Media Server here. Click [Restart Media Server] button to restart Media Server.
Media Folder
Specify the folder on the LinkStation/TeraStation that you want to designate as the Media Folder. Choose a folder that contains your media files. To use a folder located on an external USB drive connected to the LinkStation/TeraStation, put a checkmark next to [USB Disk].

[] Permit access to DLNA clients
Available DLNA Clients
Available media players' MAC addresses, IP addresses and device names are displayed. Check the [Allow Access] check box and click [Apply] for each media player that you want to allow to access your media files. To block a media device from accessing the media located on your TeraStation/LinkStation, uncheck its box. Click [Refresh client list] to update the list of obtain DLNA media players.

[] Database refresh setting
Automatic Update
Specify whether or not the DLNA Database updates automatically. [Refresh] automatically updates the list for you; [Remain] keeps the old list and does not add new DLNA media players as they become available.
Check [Refresh now] and click [Apply] to update the list of media players immediately.
Check [Initialize database before refresh] if you want the LinkStation/TeraStation to initialize its database before each refresh
Refresh interval
You may specify the automatic update interval time in minutes.

[] Squeezebox Server
You may specify the automatic update interval time in minutes.
The Squeezebox is a network music player by Logitech that can play back music stored on your network using your wired or wireless LAN. Enable the Squeezebox Server to play music on the LinkStation with the Squeezebox. No computer is needed. Follow the procedure below to configure.

1. In the Web Admin interface, navigate to [Extensions]-[MediaServer]-[Modify Settings].
2. Enter the [Port No.] for the Squeezebox Server. The initial setting is 9001. Normally, this does not need to be changed.
3. Click [Open Squeezebox Server Settings]. This displays the setting screen for the Squeezebox Server.
For more information on how to use the Sqeezebox Server, do an Internet search for "Squeezebox Server".

Note:
The following file types are supported.
Music Files mp3, flac, ogg, he-aac v2, wma

This completes the procedure for enabling playback of music data saved in the LinkStation.
Some of LinkStation models does not support Squeezebox Server.
