[] Disk Backup
Disk Backup
Up to 8 scheduled backup jobs can be created. Click on [Create New Job] to configure a new backup job. You may schedule your LinkStation to create a backup to an external USB hard drive or to another LinkStation.

To use a USB hard drive as your backup destination, its file system must be either FAT32, XFS, or EXT3.
FAT32 has a maximum file size of 2GB. XFS or EXT3 is recommended for backup hard drives.
Never unplug the power or USB cord while performing a backup.

You cannot perform the following operations during the backup process:
- LinkStation Initialization
- Disk format
- Disk check
- Create backup job
- Addition/editing/deletion of shared folder/group/user

[] Backup Destination Shared Folder Password
To create a backup job to a share that is password protected as a backup destination, enter the password to see protected shares with that password.

[] Edit Backup Job
Backup Job Schedule
Select from "Not Scheduled", "Immediately", "Every day", "Every week".
The backup job will not run if you choose "Not Scheduled".
Backup Date
Select the day you want the backup to occur.
Backup Time
Select the time you want the backup to occur.
Backup Function Mode
Backup mode selection:
*Normal Backup makes a complete backup.
*Overwrite backup options modify an existing backup.
*Overwrite backup (Complete/Differential) updates the previous backup with new files and changed files.
*Overwrite backup (Append) updates the previous backup with new files only.
*Overwrite backup (Differential) updates the backup with modified files only.

Caution: Overwrite backup (Differential) option will write over modified files from previous backup.
Caution: Do not access any backup file(s) during the backup. This may cause an error.
Backup Option
[Create a folder inside of the backup target folder.]
This option creates a new folder for the backup inside the original backup target folder.
[Use Encryption]
If selected, this option will use an encrypted connection between the source LinkStation and the backup LinkStation for secure data transfer. Transfer speed will be reduced when using encryption. Encryption is not available if the backup destination is a USB hard drive.
[Compressed Transfer]
You may enable compression for data transfer. Compression may increase transfer speeds on networks with limited bandwidth. Do not enable compression if the backup destination is a USB 2.0 hard drive.
[Exclude trash boxes from backup target]
If this option is checked, the Trashbox folder will not be backed up.
[Create backup log file]
This creates a log file of the backup job.
[Ignore backup failure and continue backup job on schedule]
If enabled, future scheduled backups will continue even if there are errors with this backup.

[] LinkStation List
LinkStation List
Display list of LinkStations on the network.
Click [Renew] to refresh the list.

[] Add Remote Off-Subnet LinkStation
Remote LinkStation IP Address
To use a LinkStation from your subnet that is not displayed on the list, or to add a LinkStation from a different subnet, enter the LinkStation's IP address and click [Add to List].
To delete the registered LinkStation from the list, select the LinkStation and click [Remove from List].
