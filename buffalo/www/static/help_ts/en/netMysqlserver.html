[] MySQL Settings
Click [Modify Settings] to change settings.

[] MySQL Server
Enable to use the MySQL server.

[] Port Number
Choose a port for MySQL.

[] Data Folder
Choose a shared folder that the MySQL database will be saved in.

[] phpMyAdmin
phpMyAdmin lets you easily manage a MySQL database. This program is included as a convenience. Note that it is neither written nor supported by Buffalo Technology.

Enable the web server before using phpMyAdmin.

By default, log in to phpMyAdmin with the following account:

Username: admin
Password: password

After logging in, you can change the user name and password from within phpMyAdmin.

Note: The phpMyAdmin account's username and password are not linked to the TeraStation's admin user name and password. They are separate.
