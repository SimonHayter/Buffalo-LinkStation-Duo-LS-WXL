[] Media Server
Media Server Settings
To change settings, click [Modify Settings]. If your media player requires a media server to play media from your TeraStation, then enable Media Server here. Click [Restart Media Server] button to restart Media Server.

[] Media Folder
Specify the folder on the TeraStation that you want to designate as the Media Folder. Choose a folder that contains your media files. To use a folder located on an external USB drive connected to the TeraStation, put a checkmark next to [USB Disk].

[] Permit access to DLNA clients
Available DLNA Clients
Available media players' MAC addresses, IP addresses and device names are displayed. Check the [Allow Access] check box and click [Apply] for each media player that you want to allow to access your media files. To block a media device from accessing the media located on your TeraStation, uncheck its box. Click [Refresh client list] to update the list of obtain DLNA media players.

[] Database refresh setting
Automatic Update
Specify whether or not the DLNA Database updates automatically. [Refresh] automatically updates the list for you; [Remain] keeps the old list and does not add new DLNA media players as they become available.
Check [Refresh now] and click [Apply] to update the list of media players immediately.
Check [Initialize database before refresh] if you want the TeraStation to initialize its database before each refresh
Refresh interval
You may specify the automatic update interval time in minutes.
