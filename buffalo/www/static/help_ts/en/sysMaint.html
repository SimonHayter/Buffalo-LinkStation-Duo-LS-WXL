[] Email Notification

You may set up email alerts for the following occurrences:.
- HD Status Report
- Fan errors
- Disk errors
- Backup Complete
- Network errors
- Quota Reports
- Log errors
- Replication Reports

SMTP Server
Enter the IP address and port number of your SMTP mail server. If no port number is entered, the default port 25 will be used.

POP3 Server
Many types of SMTP mail servers and authentication may be used with the TeraStation. Consult your IT department or ISP for any necessary login information to use your SMTP server.

Subject
Specify a subject line for the email notifications, such as "TeraStation Status Report".

Recipient Mail Address
Notification emails can be sent to multiple email addresses.

[] Syslog Transfer

Forwarding Log Messages
You may configure the TeraStation to forward logs to a syslog daemon or server.

[] Alert Sound Settings

Mail Notification
You may set up sound alerts for the following errors:.
- Overheating
- Disk Error
- Fan Error
- UPS Power Error

[] Restart TeraStation

To reboot the TeraStation, click the [Restart] button.

[] Shutdown TeraStation

To turn the TeraStation off, click the [Shutdown] button.
