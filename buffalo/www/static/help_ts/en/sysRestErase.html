[] Restore Factory Defaults
Clicking [Restore TeraStation] restores your TeraStation to original factory settings. You may choose the behavior of the reset button with regard to the Admin Password on the TeraStation. If [Keep current admin password] is selected, then the admin password will not be reset. If [Restore admin password to factory defaults] is selected, then the [Restore TeraStation] button will reset the admin password to 'password'.

[] Erase
Click [Erase] to overwrite all data on the disk with zeros. Erasing the TeraStation is recommended when you sell, donate, or give it away, or if you otherwise want to delete all data from the unit so that it can never be recovered. This cannot be undone. After erasing, all settings on the TeraStation are restored to their factory defaults.
